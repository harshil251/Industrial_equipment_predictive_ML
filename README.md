### ğŸ”¹ Anomaly Detection

  * Suitable for unlabeled and high-dimensional sensor data

* **Density-Based Spatial Clustering of Applications with Noise(DBSCAN)**
  
  * It does not require the user to specify the number of clusters in advance.
  * It is effective at identifying and handling noise/outliers by not assigning them to any cluster.
